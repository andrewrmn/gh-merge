About GitMerge

{% include patterns/pattern-about.html %}

Git Merge is the pre-eminent Git-focused conference: a full-day offering technical content and user case studies, plus a day of workshops for Git users of all levels. Git Merge 2018 takes place March 8th in Barcelona, Spain at the Convent Del Àngels at the MACBA.

Git Merge is dedicated to amplifying new voices in the Git community and to showcasing the most thought-provoking projects from contributors, maintainers and community managers around the world.
